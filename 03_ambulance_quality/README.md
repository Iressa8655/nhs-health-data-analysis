## What this analysis will cover

Ambulance Quality Indicators (AQI) data captures response times by call category, outcomes, and — critically — **hospital handover delays**: the time ambulance crews spend outside emergency departments waiting to hand patients over.

This is the direct link between ambulance performance and A&E performance. When A&E departments are full, ambulances queue outside. Each ambulance queuing is unavailable for new emergency calls. The handover delay problem is therefore simultaneously an ambulance capacity problem and an A&E flow problem.

This analysis will answer:

- Are Category 1 and Category 2 response time standards being met nationally?
- Which ambulance trusts have the worst average handover delays?
- Is there a statistical relationship between handover delay times and A&E four-hour performance at the same trusts?
- How has handover performance changed over time?

## Connection to Project 01 (A&E)

The A&E analysis identified geographic clusters of poor performance — particularly in Cheshire/North West and the Midlands. Ambulance handover data for those same regions will help determine whether long handover times are a contributing factor, or whether the A&E problems are primarily internal.
